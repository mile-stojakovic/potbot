# PotBot
The source code for The Scrabblers' ENGR 130 Project 3, PotBot.

## Setup
### Prerequisites
You should have PlatformIO and Git installed on your computer.

### Installation
Clone the repository from GitHub:
```bash
git clone https://github.com/mile-stojakovic/potbot.git
```
If you have the PlatformIO plugin for VSCode, you should just be able to click the compile and run buttons. If you are using the CLI version of PlatformIO, Navigate into the folder and run the following commands:
```bash
pio run
pio run -t upload
```
